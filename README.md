# パソコン太郎 新入社員サポートBOT

## 📋 概要
パソコン太郎の新入社員が会社の事業内容や業務について気軽に質問できるチャットボットシステムです。

## 🏢 パソコン太郎について
- 公式サイト: https://pasotaro.com/
- 主な事業: 講演会企画運営、児童向け絵本制作・販売、IT教育・職業教育、地域連携
- 拠点: 栃木県を中心に活動

## ✨ 主な機能

### 1. チャット形式の対話
- リアルタイムでの質問と回答
- 自然な会話形式のインターフェース

### 2. よくある質問ボタン
以下のトピックについてワンクリックで質問できます:
- 🏢 会社概要
- 🎤 講演会
- 📖 絵本事業
- 🎓 教育事業
- 🤝 地域連携
- 🎪 イベント運営

### 3. 対応可能な質問カテゴリ
- **会社概要**: 事業内容、ビジョン、ミッション
- **講演会・イベント**: 企画運営、過去実績（堀江貴文氏、茂木健一郎氏など）
- **絵本事業**: 夢絵本シリーズ、職育シリーズ、Amazon販売
- **教育事業**: IT教育、職業教育、リテラシー向上
- **地域連携**: 栃木県との協働、環境省連携、社会貢献
- **イベント運営**: 当日業務、会場設営、受付対応
- **絵本販売**: 購入方法、物流、寄贈活動
- **SNS・広報**: 発信内容、運用ルール
- **業務の流れ**: タスク管理、作業手順
- **連絡先**: 各担当者への問い合わせ

## 🚀 使い方

### ローカルで実行
1. このディレクトリで以下のコマンドを実行:
```bash
python3 -m http.server 8000
```

2. ブラウザで開く:
```
http://localhost:8000
```

### 質問方法
1. **クイックボタンを使用**: 「よくある質問」から選択
2. **直接入力**: チャット欄に質問を入力して送信

## 🔧 カスタマイズ方法

### 新しい質問を追加する
`script.js` の `knowledgeBase` オブジェクトに以下の形式で追加:

```javascript
'新しいトピック': {
    keywords: ['キーワード1', 'キーワード2'],
    answer: `回答内容をここに記述`
}
```

### クイックボタンを追加する
`index.html` の `.quick-buttons` セクションに以下を追加:

```html
<button class="quick-btn" onclick="askQuestion('質問内容')">
    🔹 ボタン名
</button>
```
